# Questionnaire Generator Frontend

## Project setup

### Compiles and hot-reloads for development

```
make
```

### Compiles and minifies for production

```
make build-production
```

### Run your unit and end-to-end tests

```
make tests
```

### Lints and fixes files

```
make lint
```
## Default Url
```
http://localhost:8080/
```
